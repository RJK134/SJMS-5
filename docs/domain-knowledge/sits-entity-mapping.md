# SITS Entity Mapping Reference

STU (Student) = Person + Student + PersonName/Address/Contact
SCJ (Student Course Join) = Enrolment
SCE (Student Course Enrolment) = ModuleRegistration
SPR (Student Programme Route) = ProgrammeRoute
SMO (Student Module Outcome) = ModuleResult
SMR (Student Mark Record) = AssessmentAttempt then ModuleResult
SQA (Student Qualification Aim) = Award
CAP (Course Admission Programme) = Application then Programme
ACD (Admission Course Decision) = Application.status + OfferCondition
CAM (Course Assessment Module) = Assessment
