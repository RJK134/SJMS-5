# Student Lifecycle Process Map

ENQUIRY then APPLICATION then OFFER then ACCEPTANCE then ENROLMENT then REGISTRATION
then Module Selection then TEACHING/LEARNING then ASSESSMENT/MARKS then EXAM BOARD
then PROGRESSION DECISION (PROGRESS or REPEAT or WITHDRAW)
then FINAL YEAR then AWARD/GRADUATION then ALUMNI

Key State Machines:

Application: submitted, under-review, interview, conditional-offer, unconditional-offer, firm, insure, declined, withdrawn, rejected

Enrolment: enrolled, interrupted, suspended, withdrawn, completed, transferred

Assessment Attempt: pending, submitted, marked, moderated, confirmed, referred

Mark Pipeline: DRAFT, FIRST_MARK, SECOND_MARK, MODERATED, EXTERNAL_REVIEWED, BOARD_APPROVED, RELEASED
