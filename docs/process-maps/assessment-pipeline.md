# Assessment Pipeline

Module Leader creates Assessment (type, weight, due date, rubric)
Student submits Work (file to MinIO, Turnitin check)
First Marker enters Mark + Feedback
Second Marker double-marks sample (threshold: above 70, below 40, borderline)
Module Leader moderates (reviews sample, adjusts if needed)
External Examiner reviews sample + moderation
Exam Board confirms marks, progression decisions
Registry releases to student portal

Grade Boundaries (UK Standard):
70+ = First
60-69 = Upper Second (2:1)
50-59 = Lower Second (2:2)
40-49 = Third
Below 40 = Fail

Degree Calculation: Year 2 weight 33 percent, Year 3 weight 67 percent (typical). Best 100 credits at each level, or all credits, use better outcome for student.
