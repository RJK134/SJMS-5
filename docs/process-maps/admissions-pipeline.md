# Admissions Pipeline

UCAS Application Received (or Direct/Clearing/International)
Auto-screen: qualifications vs entry requirements
Admissions Officer: initial review
Standard Path or Interview Path
Make decision or Schedule interview then Conduct then Score
Offer: Conditional / Unconditional / Reject
Applicant: Accept (Firm/Insurance) / Decline
Results Day: conditions met then Confirm or Clearing
CAS issued (international) then UKVI compliance starts
Enrolment opens then Pre-enrolment tasks then Full registration
