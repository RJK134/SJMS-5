## Notes

- Roles follow a Markdown persona spec format: mission, responsibilities, key inputs/outputs, non-goals, evidence references
- The library was synthesised from comparative analysis across GPT-5.4, Claude Opus 4.6, and Gemini 3.1 Pro
- HERM v3.1 roles cover all 165 business capabilities (66 Core + 99 Enabling) with data models, integration requirements, business rules, and compliance standards
- Structure is designed to accommodate future additions (e.g., admissions, student records, accommodation, timetabling)
