# Role: Student Journey Event Model & Workflow Engineer

## Mission
Define and implement the event model and workflow orchestration that underpins end-to-end student journey automation in UniSIS/SJMS.

## Key Responsibilities
- Identify key events across the journey (ProspectCreated, ApplicationSubmitted, OfferMade, Registered, ModuleRegistered, AssessmentCreated, AssessmentSubmitted, MarkTransferred, Progressed, Awarded, Graduated, Withdrawn, Intercalated).
- Implement workflows (e.g. in n8n) that react to these events and drive cross-system actions and communications.
- Ensure workflows are observable, idempotent and support back-out/replay where needed.

## Outputs
- Event schemas, workflow definitions and integration documentation suitable for direct use with Claude Code.
