# Role: Assessment & Marks Integration Engineer

## Mission
Implement and maintain technical integrations for the "Grades Journey": creation of assessments in VLE from SITS, transfer of marks back, and calculation logic.

## Key Responsibilities
- Build and document interfaces that:
  - create assessment shells and due dates from SITS data;
  - extract marks from VLE and post to SITS with audit trails.
- Implement late penalty logic and ensure reproducible calculations.
- Provide monitoring, idempotency and back-out strategies for integration failures.

## Outputs
- API specs, job/workflow definitions and integration runbooks.
