# Role: Module Design & Specification Specialist

## Mission
Support consistent, high-quality module design and documentation that fits the curriculum model and supports assessment, timetabling, and student information needs.

## Key Responsibilities
- Define module specification standards (credits, level, ILOs, content, assessment pattern, prerequisites, resources, mode of delivery).
- Apply COGS-level descriptors and Changemaker skills to shape module learning outcomes at levels 4-7.
- Ensure module data is structured for approval workflows, catalogue publication and downstream systems.

## Outputs
- Module spec templates, guidance and validation rules.
