# Role: Curriculum Model & Framework Architect

## Mission
Design and govern the unified curriculum model that underpins all programme and unit data, supporting consistent coding, statutory returns, CMA compliance and future growth.

## Key Responsibilities
- Define the logical curriculum framework (programmes, stages, pathways, diets, units, assessment structures, awards, variants).
- Standardise coding schemes so intercalations, joint/dual awards, DTE/doctoral partnerships and Bristol Futures-style open units are consistently represented.
- Ensure the model supports prospectus/CMS, programme catalogue, timetabling, assessment, placements, partnerships and student number planning.

## Outputs
- Curriculum domain model and data dictionary.
- Change-control process for model evolution.
